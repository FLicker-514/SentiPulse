# SentiPulse 数据采集说明

## 一、数据来源

共 **4 个数据源**：A股日线行情、东方财富财经新闻、巨潮资讯公司公告、证券时报新闻。

### 1. 股票数据 — A股日线行情

| 项目 | 说明 |
|------|------|
| **数据来源** | Baostock (证券宝) |
| **数据类型** | A股日线 K 线数据 (OHLCV)，前复权(qfq)/后复权(hfq)/不复权(none) |
| **覆盖范围** | 2025-01-02 ~ 2026-06-05 |
| **数据量** | 10,290 条（10股 × 343交易日 × 3复权） |

> AKShare 接口不稳定，全部使用 Baostock 采集。Baostock 是证券宝提供的免费 A 股历史数据接口，数据与上海证券交易所官网一致。三种复权方式均已采集，互有差异。

### 2. 财经新闻 — 东方财富 (Eastmoney)

| 项目 | 说明 |
|------|------|
| **数据来源** | 东方财富新闻搜索 API |
| **数据类型** | 聚合自证券时报、中国证券报等权威财经媒体的新闻报道 |
| **覆盖范围** | 2025-12 ~ 2026-06（约6个月） |
| **数据量** | 5,230 篇 |

> Human Review：仅爬取简略信息，未获取全文内容。部分快讯类型文章提供了全文字段，但数量有限。

### 3. 公司公告 — 巨潮资讯 (Cninfo)

| 项目 | 说明 |
|------|------|
| **数据来源** | 巨潮资讯网 (cninfo.com.cn)，中国证监会指定信息披露平台 |
| **数据类型** | 上市公司法定披露公告（董事会决议、定期报告、回购进展等） |
| **覆盖范围** | 2025-01-07 ~ 2026-06-06 |
| **数据量** | 3,417 篇 |

> Human Review：仅 4 篇无法获取 PDF 全文，其余 3,413 篇均通过 PyMuPDF 成功提取正文。约 40 篇为扫描件，OCR 识别后文本质量较差。PDF 及 TXT 保存在 `data/pdf/`，解析评估见 `data/ocr_pdf_info/`。

### 4. 证券时报 (STCN) 新闻

| 项目 | 说明 |
|------|------|
| **数据来源** | 证券时报网 (stcn.com) |
| **采集方式** | Playwright 浏览器模拟 + 人工验证码 |
| **覆盖范围** | 2024-08-14 ~ 2026-06-05 |
| **数据量** | 1,208 篇 |

> Human Review：STCN 搜索接口有 RSA 加密头 + 阿里云验证码保护，通过 Playwright 启动真实浏览器、人工完成验证码后自动滚动翻页采集。正文内容已通过详情页抓取补全。

## 四、使用方法

```bash
cd data_crawler
uv sync                              # 安装依赖
uv run python main.py               # 全量采集
uv run python main.py --stock       # 仅股票
uv run python main.py --news        # 仅新闻
uv run python main.py --status      # 查看统计
```

### 核心脚本

| 文件 | 功能 |
|------|------|
| `main.py` | 采集入口 |
| `stock/stock_crawler.py` | A股日线采集（Baostock） |
| `news/eastmoney_crawler.py` | 东方财富新闻搜索 |
| `news/cninfo_crawler.py` | 巨潮资讯公告搜索 |
| `news/cninfo_content_fetcher.py` | 公告 PDF 下载与正文提取 |
| `news/stcn_scripts/stcn_playwright_crawler.py` | 证券时报 Playwright 浏览器采集 |

### 输出

采集数据存入 `data/sentipulse.db`（SQLite），同时导出为 CSV 至 `../data/` 目录供下游使用。详细数据说明见根目录 `data_usage.md`。
