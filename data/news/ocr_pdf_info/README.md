# PDF 解析信息汇总

`data/pdf/` 目录下保存了从巨潮资讯（Cninfo）下载的上市公司公告 PDF 及 OCR 提取文本。

## 概览

| 指标 | 数值 |
|------|------|
| 覆盖企业 | 9 / 10（海尔智家暂无PDF） |
| PDF 总数 | 36 |
| 文件总大小 | ~450 MB |
| 电子排版PDF（可直接提取文本） | ~20 篇 |
| 扫描件PDF（需OCR，文本质量较差） | ~16 篇 |
| TXT 缺失 | 0 |

## OCR 质量分类

| 分类 | 判断标准 | 数量 |
|------|----------|------|
| **TEXT_GOOD** | TXT/PDF 比 > 0.01，电子排版，文本提取完整 | ~12 |
| **SCANNED_OK** | TXT/PDF 比 0.001-0.01，扫描件但OCR可识别主要内容 | ~20 |
| **SCANNED_POOR** | TXT/PDF 比 < 0.001，扫描件，OCR文本质量差（多为年报、财报） | ~4 |

> 扫描件多为年度报告、审计报告，页面多、图文混排，OCR识别效果有限。电子排版PDF多为法律意见书、股东大会决议等较短文件，文本提取完整。

## 数据来源与处理流程

1. **来源**: 巨潮资讯网 (cninfo.com.cn) — 中国证监会指定信息披露平台
2. **下载**: `cninfo_content_fetcher.py` 自动构造 PDF URL (`static.cninfo.com.cn/finalpage/{date}/{id}.PDF`)
3. **提取**: PyMuPDF (fitz) 提取文本，截取前 10,000 字符存入数据库
4. **存储**: PDF 原始文件和 TXT 文本均保存在 `data/pdf/{公司名}/` 下

## 局限性

- 扫描件（约占44%）OCR质量有限，年报、审计报告等大型扫描件的表格和数据识别不完整
- 部分PDF为混合类型（文本+扫描页），PyMuPDF对文本页提取完整，扫描页提取差
- 极少数PDF（4篇）因链接失效或格式异常无法下载
